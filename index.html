About

I have always been fascinated by the power of technology, and how we can use it to make our lives easier, more efficient, and enjoyable. I do believe though, that technology should be used with humans, and not users, in mind.

I graduated from Brigham Young University - Idaho with a Bachelor of Science in Web Design and Development, and that's where I first learned about Human Computer Interaction.

It was so interesting to me to learn the design fundamentals, tools and methods that we can use to build technology that meets the specific needs of those using the technology, that I decided to continue my studies. I am currently working towards a Master of Science in Information Technology at Brigham Young University in Provo, Utah, where I’m focusing on Human Computer Interaction.

Penny Juice Site Redesign

Redesign of the “Penny Juice” site (pennyjuice.com). Completed user research, an analysis of the results, created documentation to define the needs of the user and how to achieve a better experience, created a high-fidelity prototype, developed a new logo, coded the site, and performed user testing.

PTA Site Redesign

Redesign of the PTA site of a local school. The site was originally hosted as WordPress Blog, which was not an efficient way to share information to users. Completed a preliminary analysis, user research, defined criteria and recommendations, created a high-fidelity prototype and coded the site.

Zen Garden Redesign

Redesigned the CSS Zen Garden site (csszengarden.com) with a Candy Shop theme using only CSS and SCSS without editing classes, ids or any part of the original HTML. Included animations, SVG images created on Illustrator, and updated the layout. Work was completed with a partner.
